Jordan Measure
for $E \subset [0,1]$ define $J_*(E) = inf\left\{\sum_{j=1}^N |I_j|: E \subset \bigcup_{j=1}^N I_j\right\}$

$J^*(Q_{[0,1]}) = 1$

The measure of contable set is zero

if $X = \left\{x_1, \cdots, x_n, \cdots\right\}$
let $I_k = \left(x_k - \epsilon 2^{-k}, x_k + \epsilon 2^{-k}\right)$
then $m(X) = m(\bigcup_{k=1}^\infty I_k) \leq \sum_{k=1}^\infty m(I_k) = 2\epsilon \sum_{k=1}^\infty 2^{-k} \leq c\epsilon$
as this is true for all $\epsilon > 0$, $m(X) = 0$
(Q: $I_k$ should be an open set, however, if we take $\epsilon = 0$, then $I_k$ is a empty set)
